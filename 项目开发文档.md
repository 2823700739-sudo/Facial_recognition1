# 人脸隐私加密便签—项目开发详细文档
## 一、项目基础信息
1. 开发语言：C++11
2. UI框架：Qt 5.15.2
3. 视觉库：OpenCV 4.5.5（Haar人脸检测）
4. 数据库：SQLite3（Qt内置驱动）
5. 项目难度：入门级
6. 开发周期：3~5天
7. 运行环境：Windows 10/11
8. 核心模式：纯离线、无联网、隐私加密存储

## 二、项目创新亮点
1. 人脸识别替代传统密码，无感解锁私密便签；
2. 支持文字笔记+屏幕截图双内容加密存储；
3. 3秒无人脸自动锁定，防偷窥隐私泄露；
4. 仅存储人脸特征值，不保存人脸原图，合规安全；
5. 采用SQLite3结构化存储，数据稳定易维护；
6. 全程本地运行，不上传任何隐私数据到云端。

## 三、全套技术栈&详细配置说明
### 3.1 核心开发工具
| 工具名称 | 指定版本 | 核心用途 |
| ---- | ---- | ---- |
| Qt | 5.15.2 | 界面开发、摄像头调用、定时器、数据库操作 |
| MinGW | 7.3.0 | Qt配套C++编译器 |
| OpenCV | 4.5.5 预编译MinGW版 | 人脸检测、视频画面处理、矩形框绘制 |
| SQLite3 | Qt内置原生版 | 本地轻量化数据库，存储人脸&笔记数据 |

### 3.2 Qt模块依赖
项目.pro文件必须开启：
QT += core gui widgets sql multimedia

### 3.3 OpenCV环境配置
1. 解压OpenCV到无中文路径：D:\opencv455
2. 系统环境变量添加：D:\opencv455\x64\mingw\bin
3. .pro引入依赖：
INCLUDEPATH += D:/opencv455/include
LIBS += -LD:/opencv455/x64/mingw/lib -lopencv_world455

### 3.4 SQLite3配置说明
无需额外安装软件，Qt自带SQLite驱动；
程序运行后自动在exe同级目录生成 note_db.db 数据库文件。

### 3.5 加密&辅助技术
1. 自定义异或加密：实现文字、图片Base64数据加密解密；
2. Qt Base64编码：截图转文本存入数据库；
3. QTimer定时器：实现实时人脸检测、自动倒计时锁定。

## 四、系统整体架构
分层极简架构（入门易维护）：
1. UI交互层：登录锁定页、便签编辑页、截图功能按钮；
2. 业务逻辑层：人脸注册/验证、加密解密、截图保存、自动锁定；
3. 视觉算法层：OpenCV调用摄像头+人脸框检测；
4. 数据持久层：SQLite3存取人脸特征、笔记文本、加密截图。

## 五、SQLite3数据库详细设计
### 5.1 数据库文件名
note_db.db

### 5.2 数据表1：user_face（存储注册人脸）
字段设计：
id INTEGER PRIMARY KEY AUTOINCREMENT
face_feature TEXT 加密后人脸特征字符串
create_time TEXT 人脸注册时间

建表SQL：
CREATE TABLE IF NOT EXISTS user_face (
id INTEGER PRIMARY KEY AUTOINCREMENT,
face_feature TEXT NOT NULL,
create_time TEXT NOT NULL
);

### 5.3 数据表2：note_data（存储私密便签）
字段设计：
id INTEGER PRIMARY KEY AUTOINCREMENT
text_content TEXT 加密文字笔记
img_content TEXT 截图Base64加密数据
update_time TEXT 最后修改时间

建表SQL：
CREATE TABLE IF NOT EXISTS note_data (
id INTEGER PRIMARY KEY AUTOINCREMENT,
text_content TEXT,
img_content TEXT,
update_time TEXT
);

## 六、全部功能模块拆解
### 模块1：人脸认证模块
1. 首次启动引导人脸注册，采集人脸特征存入数据库；
2. 打开软件默认锁定，实时摄像头识别人脸；
3. 匹配成功自动解锁进入便签页；
4. 支持一键重置人脸，清空原有注册信息。

### 模块2：私密便签编辑模块
1. 纯文本增删改查，实时编辑；
2. 退出锁定/关闭软件自动加密保存到数据库；
3. 解锁后自动解密展示笔记内容。

### 模块3：私密截图扩展功能
1. 一键截取电脑屏幕任意画面；
2. 图片转为Base64编码+异或加密存入数据库；
3. 人脸离开摄像头，截图预览自动隐藏防偷窥。

### 模块4：智能安全锁定模块
1. 手动点击按钮随时锁定便签；
2. 定时器实时检测：3秒无人脸自动上锁；
3. 锁定后清空笔记&截图预览，保护隐私。

### 模块5：数据库底层模块
1. 程序启动自动检测并创建数据库+数据表；
2. 封装人脸、笔记、截图的增删改查接口；
3. 所有写入数据均加密，杜绝明文存储。

QString encrypt(QString data,QString key="face2026"){
QString res;
for(int i=0;i<data.size();i++)
res+=QChar(data[i].unicode()^key[i%key.size()].unicode());
return res;
}

### 7.3 Qt屏幕截图转Base64
QString getScreenBase64(){
QPixmap p=QGuiApplication::primaryScreen()->grabWindow(0);
QByteArray b;QBuffer buf(&b);
p.save(&buf,"PNG");
return b.toBase64();
}

### 7.4 自动锁定计时逻辑
